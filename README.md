# 旅行スケジュール管理Webアプリケーション

友人との旅行スケジュール管理を効率的に行うためのWebアプリケーションです。

## 特徴

- **静的サイト**: GitHub Pagesで公開可能なフロントエンドのみの構成です。
- **データ共有**: JSON形式でのエクスポート・インポートにより、メンバー間でデータを共有できます。
- **レスポンシブ**: PC、タブレット、スマートフォンに対応しています。
- **オフライン対応**: localStorageを使用しているため、一度読み込めばオフラインでも動作します。

## 主な機能

1. **スケジュール管理**: 時系列でイベントを管理。場所との紐付けも可能。
2. **場所情報管理**: 訪問予定地の住所、営業時間、画像、リンクを管理。Google Mapsへのクイックアクセス。
3. **予算・費用管理**: 旅行中の費用を記録し、合計金額や割り勘の目安を自動計算。
4. **やることリスト**: 準備物やタスクをチェックリスト形式で管理。
5. **メンバー管理**: 旅行に参加するメンバーを設定し、支払い担当者やタスク担当者を割り当て可能。

## 使い方

1. `index.html` をブラウザで開きます。
2. 各タブからスケジュール、場所、費用などを入力します。
3. データを共有したい場合は、ヘッダーの「保存」ボタンをクリックしてJSONファイルをダウンロードし、共有相手に送ります。
4. 共有されたデータを読み込むには、「読み込み」ボタンからJSONファイルを選択します。
